# Biology Core Review

Path: `topics/biology/core`

## Scope

This topic is a self-paced biology review course. It covers the standard high-school-to-intro-college biology arc: molecules, cells, energy, signaling, heredity, gene expression, evolution, and ecology.

The local course name is simply Biology. The scope is grounded in:

- College Board's current biology course framework for the 2025-26 school year.
- Khan Academy's college biology unit map and practice structure.
- OpenStax Biology 2e as the open textbook backbone for deeper explanations.
