# Probabilistic Graphical Models Exam Prep Notes

This note is the hub for a full exam-prep note set covering the class from a catch-up perspective.

The design goal is:

- beginner friendly
- slower than lecture notes
- broad enough for exam review
- organized by module so the material is not overwhelming

## How To Use This Set

- Start with [[2026-04-20-pgm-course-overview]] if you want the big-picture map first.
- Start with Section 1 if you feel shaky on what a graphical model even is.
- If you already know the representation side, jump to Section 3 for inference.
- If your exam is likely to emphasize later material, Sections 5 through 8 are the late-course modules.
- For each topic, the intended flow is:
  notes $\to$ worked problems $\to$ practice problems $\to$ answer key $\to$ memory sheet.
- Use the sections as readable notes first. After that, use the worked problems, practice problems, and the practice exam for active recall.

## What the Companion Pages Are For

- `section notes`: the main teaching document for the topic
- `worked problems`: slower examples with solutions and explanations
- `practice problems`: unsolved questions for active recall
- `practice answer key`: concise target answers for the unsolved practice set
- `memory sheet`: the shortest high-yield facts and distinctions to memorize

If you are new to the course, the safest pattern is:

1. read the section notes slowly
2. study the worked problems until the examples make sense
3. try the practice problems without help
4. check the answer key
5. finish with the memory sheet
