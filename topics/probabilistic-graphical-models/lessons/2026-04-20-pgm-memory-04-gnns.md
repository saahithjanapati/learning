# PGM Memory Sheet 4: GNNs and Message Passing

Use with [[2026-04-20-pgm-exam-prep-section-04-gnns]].

## Shared Intuition

- PGMs and GNNs both use local message passing on graphs

## Important Difference

- BP computes probabilistic quantities under a model
- GNNs learn embeddings for a task

## Over-Smoothing

- node representations become too similar after many layers

## Over-Squashing

- too much distant information gets compressed through graph bottlenecks

## What To Remember

- same structural motif, different objective
- over-smoothing and over-squashing are different failures

## Likely Traps

- saying over-smoothing and over-squashing are the same thing
- saying GNN message passing is literally BP
