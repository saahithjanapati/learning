# Paper Notes

Use this folder for personal notes, reactions, recaps, and follow-up questions about papers, blog posts, and processed sources.

These notes are intentionally separate from public lessons and processed source files:

- Public lessons explain the material for a reader.
- Processed sources preserve extracted source content and structured summaries.
- Paper notes preserve the learner's own reactions, mental models, open questions, and durable takeaways.

## File Layout

Store notes by topic root:

```text
learning_system/paper_notes/<root>/<source-slug>.md
```

For AI papers and AI research blogs, use:

```text
learning_system/paper_notes/ai/<source-slug>.md
```

## Note Format

Each note should include:

- source title and canonical URL
- links to processed source and lesson when available
- date the note was captured
- tags
- the learner's original recap or reaction
- a normalized takeaway that makes the idea easy to review later
- optional follow-up questions

Keep the learner's original wording when they provide a recap. Add cleanup or interpretation in a separate section so the original thought is not overwritten.
